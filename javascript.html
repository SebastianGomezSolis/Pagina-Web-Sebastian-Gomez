JavaScript

En esta sección se explican eventos importantes de JavaScript y se incluye un juego tipo tablero inspirado en Pac-Man usando HTML, CSS y JavaScript puro.

¿Qué es JavaScript?

JavaScript es el lenguaje que agrega comportamiento e interactividad a las páginas web. Permite reaccionar a eventos, modificar el DOM, validar formularios, mover elementos en pantalla y construir juegos en el navegador.

Idea clave: HTML estructura, CSS da estilo y JavaScript agrega lógica e interacción.

Eventos de JavaScript

Los eventos permiten ejecutar código cuando ocurre una acción del usuario o del navegador.

Evento ¿Para qué sirve?
clickSe ejecuta cuando el usuario hace un clic sobre un elemento.
dblclickSe dispara cuando el usuario hace doble clic.
mousedownOcurre cuando se presiona un botón del mouse sobre un elemento.
mouseupOcurre cuando se suelta un botón del mouse.
mousemoveSe ejecuta cuando el puntero se mueve dentro de un área.
mouseenterSe activa cuando el puntero entra en un elemento, sin contar sus hijos varias veces.
mouseleaveSe activa cuando el puntero sale de un elemento.
mouseoverOcurre cuando el puntero pasa sobre un elemento o sus hijos.
mouseoutOcurre cuando el puntero sale de un elemento o de alguno de sus hijos.
contextmenuSe ejecuta cuando se abre el menú contextual, normalmente con clic derecho.
keydownSe dispara cuando una tecla es presionada.
keyupSe dispara cuando una tecla se suelta.
blurOcurre cuando un campo pierde el foco.
focusOcurre cuando un elemento recibe el foco.
changeSe ejecuta cuando cambia el valor de un input, select o textarea y el cambio queda confirmado.
selectSe activa cuando el usuario selecciona texto dentro de un input o textarea.
loadSe dispara cuando un recurso termina de cargarse, por ejemplo una página o una imagen.
DOMContentLoadedEn document, indica que el HTML ya fue cargado y parseado, aunque imágenes o estilos externos puedan seguir cargando.
resizeSe ejecuta cuando cambia el tamaño de la ventana del navegador.
scrollSe activa cuando el usuario desplaza la página o un contenedor con scroll.
beforeunloadOcurre antes de abandonar la página; se usa para advertencias por cambios sin guardar.
unloadSe dispara cuando la página se está cerrando o descargando.

Ejemplo básico con archivo JS externo

<button id="btnSaludar">Saludar</button>
<script src="js/javascript.js"></script>
document.getElementById("btnSaludar").addEventListener("click", function () {
    alert("Hola desde JavaScript");
});

Aquí la lógica va en un archivo aparte, lo cual coincide con la forma recomendada para este proyecto.

Eventos que más se usan

  • click para botones.
  • keydown para mover personajes.
  • change en formularios.
  • DOMContentLoaded para iniciar el programa cuando el DOM ya existe.

Conceptos aplicados en el juego

  • Manipulación del DOM.
  • Manejo de eventos con teclado.
  • Estructuras de datos con matrices.
  • Temporizadores con setInterval.
  • Detección de colisiones.
