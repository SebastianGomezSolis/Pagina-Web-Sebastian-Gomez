¿Qué ofrece este sitio?

El sitio sirve como material de estudio y repaso durante el curso. Cada página está organizada con una introducción, explicación breve, ejemplos y fragmentos de código cuando ayudan a comprender mejor el tema.

Por ejemplo: en HTML se muestra la estructura básica de un documento; en CSS se enseña cómo aplicar estilos externos; y en backend se incluyen ejemplos con Spring Boot y Thymeleaf.

Tecnologías usadas

  • HTML5 — estructura del contenido.
  • CSS3 — diseño y estilos personalizados.
  • Bootstrap 5 — componentes y diseño responsive.
  • Font Awesome — íconos de apoyo visual.
  • JavaScript — interactividad y juego tipo tablero.
  • Google Fonts — tipografía Merriweather & Source Sans 3.
