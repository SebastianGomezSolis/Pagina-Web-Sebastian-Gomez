¿Qué ofrece este sitio?

La idea del sitio es servir como material de estudio y repaso durante el curso. Cada página está organizada con una introducción, explicación breve, ejemplos y fragmentos de código cuando ayudan a comprender mejor el tema.

Ejemplo: en la sección de HTML se muestra la estructura básica de un documento; en la sección de CSS se enseña cómo aplicar estilos externos; y en backend se incluyen ejemplos sencillos con Spring Boot y Thymeleaf.

Tecnologías usadas

  • HTML5 para la estructura.
  • CSS3 para el diseño y estilos personalizados.
  • Bootstrap 5 para componentes y diseño responsive.
  • Font Awesome para íconos de apoyo visual.
