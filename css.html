CSS

CSS se usa para dar color, tamaño, posición y estilo visual a las páginas web.

¿Qué es CSS?

CSS significa Cascading Style Sheets. Permite separar el diseño visual del contenido HTML para mantener un código más ordenado y reutilizable.

Formas de aplicar CSS

1. En línea
<p style="color: red;">Texto rojo</p>
2. Interno
<style>
  p { color: blue; }
</style>
3. Externo
<link rel="stylesheet" href="styles.css">
La forma recomendada para proyectos grandes es el CSS externo, porque facilita la organización y el mantenimiento.

Selectores principales

p { color: black; }          /* selector por etiqueta */
.destacado { color: blue; }   /* selector por clase */
#titulo { font-size: 2rem; }  /* selector por id */
section p { margin: 10px; }   /* selector descendente */
a:hover { color: red; }       /* pseudo-clase */
p::first-letter { font-size: 2rem; } /* pseudo-elemento */

Box Model

Todo elemento HTML puede verse como una caja compuesta por: contenido, padding, borde y margin.

.caja {
  width: 200px;
  padding: 20px;
  border: 2px solid #000;
  margin: 15px;
}

Display y Position

display define cómo se comporta un elemento en la página: block, inline, inline-block, flex, etc.

position controla su ubicación: static, relative, absolute, fixed y sticky.

Flexbox y buenas prácticas

.contenedor {
  display: flex;
  justify-content: center;
  align-items: center;
  gap: 1rem;
}
  • Usar CSS externo.
  • Reutilizar clases.
  • Mantener nombres claros.
  • No repetir estilos innecesariamente.
  • Combinar CSS propio con Bootstrap cuando sea útil.
