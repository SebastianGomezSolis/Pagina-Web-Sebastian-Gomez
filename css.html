CSS

CSS (Cascading Style Sheets) permite separar el diseño visual del contenido HTML, controlando colores, tipografía, espaciado, posición y mucho más.

Selectores Box Model Flexbox Display & Position

1 ¿Qué es CSS?

CSS significa Cascading Style Sheets. Permite separar el diseño visual del contenido HTML para mantener un código más ordenado y reutilizable.

2 Formas de aplicar CSS

1. En línea (inline)

HTML 
<p style="color: red;">Texto rojo</p>

2. Interno (internal)

HTML 
<style>
  p { color: blue; }
</style>

3. Externo (external) recomendado

HTML 
<link rel="stylesheet" href="styles.css">
La forma recomendada para proyectos grandes es el CSS externo, porque facilita la organización y el mantenimiento del código.

3 Selectores principales

CSS 
p { color: black; }              /* selector por etiqueta    */
.destacado { color: blue; }      /* selector por clase       */
#titulo { font-size: 2rem; }     /* selector por id          */
section p { margin: 10px; }      /* selector descendente     */
a:hover { color: red; }          /* pseudo-clase             */
p::first-letter { font-size: 2rem; } /* pseudo-elemento      */

5 Modelo de caja (Box Model)

Todo elemento HTML puede verse como una caja compuesta por cuatro áreas: contenido, padding, borde y margin.

MARGIN
BORDER
PADDING
CONTENIDO
CSS 
.caja {
  width: 200px;      /* ancho del contenido */
  padding: 20px;     /* espacio interior     */
  border: 2px solid #000; /* borde          */
  margin: 15px;      /* espacio exterior     */
}

6 Display

display define cómo se comporta un elemento en el flujo de la página.

ValorComportamiento
blockOcupa todo el ancho disponible; empieza en línea nueva.
inlineSolo ocupa el espacio de su contenido; no acepta width/height.
inline-blockComo inline pero acepta dimensiones.
flexActiva el modelo Flexbox en el contenedor.
noneOculta el elemento (no ocupa espacio).

7 Position

position controla la ubicación de un elemento respecto a su contenedor o la ventana.

ValorDescripción
staticPosición normal en el flujo (por defecto).
relativeRelativa a su posición original; desplaza sin salir del flujo.
absoluteRelativa al ancestro con position distinto de static.
fixedFija en la ventana del navegador; no se mueve al hacer scroll.
stickyCombina relative y fixed; se pega al scroll.

8 Flexbox (introducción)

Flexbox es un modelo de diseño que facilita alinear y distribuir elementos dentro de un contenedor.

CSS 
.contenedor {
  display: flex;
  justify-content: center;  /* eje principal (horizontal) */
  align-items: center;      /* eje secundario (vertical)  */
  flex-wrap: wrap;          /* permite que los items se envuelvan */
  gap: 1rem;
}
justify-content controla el eje horizontal y align-items el vertical (cuando flex-direction: row, que es el valor por defecto).

Buenas prácticas de CSS

  • Usar CSS externo para proyectos de cualquier tamaño.
  • Reutilizar clases en lugar de repetir estilos.
  • Mantener nombres descriptivos para las clases.
  • No repetir propiedades innecesariamente.
  • Combinar CSS propio con Bootstrap cuando sea útil.
  • Usar variables CSS (:root { --color: … }) para colores y fuentes recurrentes.
