Fundamentos de Internet y la Web

Conceptos básicos sobre cómo funciona la comunicación en la web, desde la infraestructura de red hasta cómo una página llega desde el servidor hasta el navegador.

Internet & Web HTTP / HTTPS Modelo OSI DNS & IP

1 ¿Qué es Internet?

Internet es una red mundial de computadoras y dispositivos conectados entre sí. Gracias a esta red se puede intercambiar información, usar aplicaciones en línea, enviar correos y navegar por páginas web.

Ejemplo: cuando una persona ve un video en YouTube o envía un correo electrónico, está usando Internet para comunicarse con servidores remotos.

2 ¿Qué es la World Wide Web?

La Web o World Wide Web es un servicio que funciona sobre Internet. Está formada por páginas y sitios web enlazados entre sí, los cuales se pueden visitar mediante un navegador como Chrome, Firefox o Edge.

Las páginas web se construyen principalmente con HTML, se diseñan con CSS y pueden agregar interactividad con JavaScript.

3 Diferencia entre Internet y Web

InternetWeb (World Wide Web)

Analogía: Internet sería la carretera, mientras que la Web son los sitios y páginas que se recorren por esa carretera.

5 Dirección IP

Una dirección IP identifica a un dispositivo dentro de una red. Permite saber quién envía datos y quién debe recibirlos.

  • IPv4: formato tradicional como 192.168.1.10.
  • IPv6: formato más largo y moderno con muchas más direcciones disponibles, como 2001:0db8::1.

6 DNS (Domain Name System)

El sistema DNS traduce los nombres de dominio a direcciones IP. Así, una persona escribe google.com en lugar de memorizar números.

El navegador pregunta al DNS cuál es la IP del dominio y luego se conecta al servidor correcto. Sin DNS tendríamos que memorizar direcciones numéricas para cada sitio.

7 Servidores, clientes y navegadores

Cliente

El programa que hace una solicitud, como un navegador web.

Servidor

La máquina que almacena recursos y responde a las solicitudes.

Navegador

Interpreta HTML, CSS y JS para mostrar la página al usuario.

8 Funcionamiento de una petición web

  1. El usuario escribe una URL en el navegador.
  2. El navegador consulta el DNS para encontrar la IP.
  3. Se envía una petición HTTP o HTTPS al servidor.
  4. El servidor responde con los recursos solicitados.
  5. El navegador renderiza la página.

9 HTTP, HTTPS, SSL y TLS

HTTP es el protocolo de comunicación usado para transferir páginas y recursos web.

HTTPS es la versión segura de HTTP. Protege la información mediante cifrado.

SSL y TLS son las tecnologías de seguridad; hoy en día TLS es la más utilizada. Su función es cifrar la comunicación entre cliente y servidor.

En una tienda en línea siempre se debe usar HTTPS para proteger datos como contraseñas o información de pago.
